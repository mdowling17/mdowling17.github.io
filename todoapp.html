To-Do App with File System

My CS50 Final Project

For my CS50 final project I created a To-Do App. My goal was to emulate the look and feel of writing an indexed list in a word processor while incorporating some additional functionality for organizing and updating your todo lists. I developed this project as a web app using React JS and React Router v6 to navigate between pages. I used a noSQL database, Firebase's Cloud Firestore, to manage the data. I also used Firebase's Authentification feature for my user login system.
