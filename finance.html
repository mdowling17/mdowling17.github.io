Mock Stock Trader

made with Python, Flask, SQLite3, the IEX Exchange API, and Bootstrap

This is another project I created when I audited CS50, Harvard's Intro to Computer Science course. The goal was to create a mock stock trading app using Python. I implemented multiple pages with routing and a user authentication system with session cookies using Flask. I also used SQLite3 (a SQL database) to store each user's portfolio, updating with sales and purchases. The website can request stock prices from IEX's exchange API. Lastly, I styled the website using Bootstrap.

Below is a screenshot of the website.

The main page contains the user's portfolio. From here you can buy and sell stocks, add funds, get a quote of stock prices, see your trade history, and manage your account by changing your password, and logging in and out.
